# AI-Surveillance-System
AI-Based real-time surveillance system using YOLOv8 and OpenCV

AI-Powered Real-Time Surveillance System

1). Overview
This project is a real-time surveillance system that detects human activity and potential threats using AI.

2). Features
- Person detection and tracking using YOLOv8
- Loitering detection
- Abnormal speed detection
- Crowd monitoring
- Restricted zone detection
- Weapon detection (knife)
- Risk scoring system with threat classification

3). Tech Stack
- Python
- YOLOv8 (Ultralytics)
- OpenCV

4). How to Run
pip install ultralytics opencv-python
python your_file_name.py
